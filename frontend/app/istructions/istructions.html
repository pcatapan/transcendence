Game Online
Game Online is an online game where players compete in Pong, moving their paddle up and down using the arrow keys. The first player to reach 10 points wins the match.
Tournament
AI Opponent
AI Opponent is a local game where players compete against an AI opponent in Pong, moving their paddle up and down using the arrow keys. This mode is used for training. The first player to reach 10 points wins the match.
1vs1 Local
1vs1 Local is a local game where two players compete against each other in Pong, using the same keyboard. The first player uses the arrow keys, while the second player uses the W and S keys. The first player to reach 10 points wins the match.
